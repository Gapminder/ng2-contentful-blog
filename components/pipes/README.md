## Pipe Date

### Usage
```typescript
import { ToDatePipe } from 'ng2-gapminder/ng2-gapminder';
// or
import {ToDatePipe} from 'ng2-gapminder/components/pipes/to-date.pipe';
```

### Annotations
```typescript
@Pipe({
  name: 'gmToDate'
})

```
